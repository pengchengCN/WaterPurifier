hexo安装说明

2018-01-01

安装 hexo

$ npm install hexo-cli -g

初始化 hexo

$ hexo init blog

$ cd blog

安装 modules

$ npm install

启动 hexo

$ hexo server

可选主题

选择主题进入github下载和配置

注意： 主题下载至 /themes

全局配置 /_config.yml

注意：/_config.yml theme: 主题名
如果github创建的项目名不是github用户名，root要配置项目名 /项目名/

主题配置 /themes/选择的主题/_config.yml

注意图片路径问题

创建归档、分类、标签、关于

$ hexo new page ‘archives’,

$ hexo new page ‘categories’,

$ hexo new page ‘tags’,

$ hexo new page ‘about’

注意：/source 文件夹会创建相应的文件夹

新建文章

$ hexo new “文章名称”
注意：新建在/source/_posts/文章名称.md
